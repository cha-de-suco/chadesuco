Introducao ao CSS

O CSS é uma linguagem essencial para os desenvolvedores web, pois permite personalizar a aparência dos sites, controlando detalhes como cor, fonte e alinhamento do texto. É crucial para a criação de sites visualmente agradáveis e com consistência visual. CSS é compatível tanto no HTML quanto no XML

Seletores Básicos

Seletor por tag: Este seletor seleciona o nome da tag HTML, por exemplo, input no CSS corresponderá a tag <input>.

                        
    HTML:
        <input>

    CSS:
        input {
            color:aqua
        }

Seletor por classe: Seleciona todos os elementos que tenha um classe. As classes são definidas em um elemento HTML e estilizadas no CSS, por exemplo:

                        
    HTML:
        <header class="cabecalho">cabeçalho</header>


            
    CSS:
        .cabecalho {
            font-size: 2em
        }

Seletor por ID: Quase a mesma coisa de seletores com classe, as únicas coisas é que um id se clara de maneira diferente, é único(só pode ser usado em um elemento, diferente de classes) e é chamado de maneira diferente no CSS:

                        
    HTML:
        <header id="cabecalho">cabeçalho</header>

    CSS:
        #cabecalho {
            font-size: 1.75em
        }
Box-Model

O Box Model é um conceito fundamental no CSS que descreve como os elementos são representados e renderizados na página.

No CSS, cada elemento é representado como uma caixa retangular, e essa representação é conhecida como Box Model. Cada caixa possui quatro componentes ou “camadas” que a envolvem, que são:

  • Conteúdo (Content): Esta é a área onde o conteúdo real do elemento, como texto, imagens ou outros elementos, está localizado.
  • Preenchimento (Padding): É o espaço entre o conteúdo do elemento e sua borda. O preenchimento é transparente, então o fundo do elemento se estende até a borda do preenchimento.
  • Borda (Border): É a linha que envolve o preenchimento e o conteúdo.
  • Margem (Margin): É o espaço ao redor da borda do elemento. Serve para separar o elemento dos outros elementos ao seu redor.

Flex-box e Grid-templates

Flexbox é um sistema de layout unidimensional que permite que você crie layouts mais flexíveis e eficientes em termos de espaço em relação aos modelos de layout tradicionais. Com o Flexbox, você pode alinhar, distribuir e organizar elementos em uma linha (ou coluna), o que é ideal para layouts responsivos e componentes de interface de usuário que precisam se ajustar a diferentes tamanhos de tela e dispositivos.

Container e Itens Flex, Eixo Principal e Cruzado, Justificação e Alinhamento são os princiapais conceitos do Flex-box

Flex-box pode ser aplicado a um container ou tag pai usando a classe, o id, ou a tag.

                    
    CSS:
        main {
            display: flex      flex-box aplicado em main
            flex-direction: row    organiza os elementos em linha ou coluna
            flex-wrap: wrap    quebra de texto se passar do limite 
        }



Grid-templates

Grid Layout é um sistema de layout bidimensional que permite que você crie layouts complexos e alinhados com precisão, dividindo a página em linhas e colunas. Ele fornece controle total sobre o posicionamento e o dimensionamento dos elementos da página, o que é especialmente útil para criar designs sofisticados e multidimensionais.

Seus principais conceitos incluem Container de Grid e Itens de Grid, Linhas e Colunas, Áreas de Grid

Grid é usado da seguinte maneira:

                    
    CSS:
        body {
            display: grid      Grid iniciado com sucesso
            grid-template-areas: 
                "header  header  header"
                "aside  main  main"
                
                body é divido em 3 colunas e 2 linhas, com cada parte sendo 
                ocupada pelo seu respectivo dono
            
        }
